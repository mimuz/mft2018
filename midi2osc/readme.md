# midi2osc

缶たたき機(MFT2018)用 MIDI→OSC変換Bridge

MIDI to OSC bridge for CanAttacker 2018 (MFT2018 Version).

# Usage

> $ cd midi2osc
> $ npm i
> $ node ./midi2osc.js
